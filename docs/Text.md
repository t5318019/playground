## Unicode

世界上每個文字都對應到 Unicode 的一個碼點 (code point)，而每個碼點可以使用 UTF-8, UTF-16, UTF-32 編碼。換句話說，Unicode, UTF-8, UTF-16, UTF-32 這四個是不同的東西，意義上相同，但不完全相等，例如 🍎 可以是「蘋果」、「apple」，但 "蘋果" 不等於 "apple" 啊。
